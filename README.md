# PowerBI-Inventory-Dashboard
Inventory dashboard in Power BI with KPIs, Stock_Status, and visuals for portfolio.

## Project Overview
- Tracks **190 products** across multiple categories and warehouses.
- Includes **calculated Stock_Status**: Low / OK / Overstock.
- Visualizes:
  - **KPIs**: Total Stock Value, Low Stock Count, Average Monthly Usage
  - **Bar chart**: Stock Quantity by Product, color-coded by Stock_Status
  - **Table**: Detailed product information with Stock_Status highlighting
  - **Optional slicers** for interactive filtering (Category / Warehouse)

## Skills Demonstrated
- Power BI Desktop  
- Data modeling & calculated columns (DAX)  
- Conditional formatting  
- Dashboard design & layout
